# 必看 | Key Point
1. 关闭 Windows 的 [阻止可能不需要的应用](https://jingyan.baidu.com/article/5d368d1ee397de7e60c057e1.html) 和关闭 [内存完整性](https://www.php.cn/faq/1516334.html)，以及关闭所有杀毒软件，这些安全措施可能会导致 LALC 无法正常启动；
2. LALC 正常启动后，有两个窗口，一个是黑框，另一个是图像交互界面，如果出现如 “无法连接” “服务器超时”，就是因为其中有一个没有启动，这种异常很可能是因为上面的安全措施没有关闭；
![教程](/img/tutorial_two_windows.png)

3. 游戏语言必须为“英语”，否则会自动停下
4. 每周四，上周如果有地牢没打完，请手动清了再启动程序，程序不知道你是否要接受这个记录
